# ai-sec-slides-template


```shell
# Собрать презентацию
make build

# Запустить в режиме обновлений по мере изменения содержимого
make watch

# Очистить от вспомогательных файлов
make clear
```

### Редактирование
Название презентации, имя автора и так далее настраивается в файле config.tex.
